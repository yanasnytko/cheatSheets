Cheat Sheet de HTML

HyperText Markup Language

<html>, <head>, <body>,

,

, ...

Il s'agit d'un "langage de balisage", du squelette de la page web, qui utilise des balises (voir plus haut). Elles indiquent l'endroit de la page où votre contenu doit être inséré, comme un coup de marqueur sur une carte.

Le squelette d'une page HTML 

<!doctype html>
<html lang="fr">
  <head>
    
     Je suis le titre 
  </head>
  <body>
    
 Je suis le contenu de la page.

  </body>
</html>

Le nom de la balise se met entre <>, ce nom est définit dans les spécifications HTML, on ne met pas ce que l'on veut.

Bases

<!doctype html>

Obligatoire, indique au navigateur quel version d'HTML on utilise sur cette page.

<html lang="fr">

C'est elle qui ouvre et ferme tout le code. Il est obligatoire d'ajouter les balises <head> et <body> entre les balises <html>.

La balise <head>

Elle détermine l'entête du document et contient les méta-informations.

Le titre de ma page HTML

Permet de donner un titre à la page.

Elle sert notamment à définir quel set de caractères la page utilise, le nom de l'auteur de la page, des mots-clés, une description, ...

Permet de lier notre page de style à la page html.

La balise <body>

C'est elle qui contient le contenu de la page. On y met toutes les balises de textes, d'images, de liens, ...

Certaines balises peuvent se voir attribuer des attributs. Ils permettent de préciser certains paramètres.

Les commentaires

Je suis un texte visible <!-- je suis un commentaire invisible -->

Les commentaires dans le code sont une bonne pratique à avoir. Ils permettent de donner des informations complémentaires à ceux qui retravaillerons dans votre code.

Balise sémantiques

article, aside, details, figcaption, figure, footer, header, main, mark, nav, section, summary, time

Le but de ces balises est de donner au navigateur et au développeur une description du contenu qui suit.

La balise <div>

La division est un conteneur. Cette balise permet de structurer sa page en "bloc".

La balise <p>

P pour paragraphe, c'est la balise la plus courante pour rédiger du texte. C'est un bloc qui est suivis d'un saut à la ligne. On peut mettre la balise <p> dans un <div> mais jamais de <p> dans un autre.
