Cheat Sheet de CSS

CSS (Cascading Style Sheets) 

.main-title {
  font-size: white;
}

Les feuilles de style en cascade permettent la mise en forme de la squelette HTML.
Le sélecteur : le nom de la balise, de classe ou d'ID auquel on veut attribuer un style. (.main-title)
La propriété : va permettre la mise en page du contenu. (font-size)
La valeur : la valeur que l'on souhaite apporter à la propriété. (white)

Interne 

<head>
  
</head>

L'avantage c'est que l'on peut appliquer une règle sur tous les mêmes éléments du document.
L'inconvénient c'est que ça ne s'applique qu'au document et non pas au site entier. Il faudra donc modifier le CSS de chaque page en cas de changement.

Externe 

<head>
  
</head>

On insère un lien vers un fichier .css de la même façon qu'on insère une image et un lien. L'avantage c'est bien la centralisation de tous vos styles et la facilité à la modification.

Les commentaires 

h1 {
  color: red;
  /* C'est rouge, parce que j'aime le rouge */
}

On peut insérer des commentaires pour soi ou pour toutes autres personnes qui lira le code. Pour ce faire on utilise un slash et un astérisque /* mon commentaire */.

Typographie 

body {
  font-family: Arial, Helvetica, sans-serif;
}

Définit la famille de police d'écriture à utiliser. 

body {
  font-size: 18px;
}

Définit la taille (hauteur) d'une police. On peut utiliser plusieurs unités (px, em,...) 

body {
  font-weight: bold;
}

Définit l'épaisseur de notre police. 

body {
  font-style: italic;
}

Permet de mettre les élements en italique.

Typographie 2 

body {
  line-height: 1.5;
}

Permet de gérer l'interlignage. 

p {
  color: #22E356;
}

Définit la couleur de l'élément. Prends un valeur hexadécimale ou un mot-clé (red, gray,...). 

p {
  text-align: right;
}

Permet d'aligner le texte, les valeurs sont: left, right, center, justify (comme sur Word).
